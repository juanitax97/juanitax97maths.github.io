## Exercice

Soit  $f(x) = x^2 - 3x + 2$.  
1. Calculer les racines de $f(x)$.  
2. Étudier le signe de $f(x)$.

---

## Correction

1️⃣ Les racines sont les solutions de $x^2 - 3x + 2 = 0$.  
$$(x - 1)(x - 2) = 0 \Rightarrow x = 1 \text{ ou } x = 2$$

2️⃣ Étude du signe :  
- \( f(x) > 0 \) pour \( x < 1 \) ou \( x > 2 \)  
- \( f(x) < 0 \) pour \( 1 < x < 2 \)
